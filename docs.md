# Frontend

The frontend is built using **Next.js**, **React**, **Tailwind CSS**, **Wagmi**, and **RainbowKit**.

## Features

* Wallet Connection
* User Dashboard
* Referral Tree
* Matrix Visualization
* Earnings Dashboard
* Token Wallet
* NFT Gallery
* Notifications
* Analytics
* Leaderboard
* Admin Dashboard

## Technology

* Next.js
* React
* Tailwind CSS
* Wagmi
* RainbowKit
* Viem
* React Query
* Axios

Run the frontend:

```bash
npm install
npm run dev
```
